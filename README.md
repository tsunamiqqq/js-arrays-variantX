# Практична робота №7
Варіант №1

## Тема
Масиви — методи, ітерація, алгоритми обробки даних.

## Мета роботи
Навчитися працювати з масивами в JavaScript, використовувати вбудовані методи, реалізовувати алгоритми сортування, фільтрації та аналізу даних.

## Опис проєкту
У цьому проєкті реалізовано систему управління студентами.  
Програма дозволяє:
1) додавати студентів
2) редагувати дані студентів
3) видаляти студентів
4) фільтрувати студентів за факультетом
5) фільтрувати за мінімальним GPA
6) сортувати студентів за GPA, ім’ям і віком
7) переглядати статистику
8) запускати Bubble Sort і Quick Sort
9) порівнювати алгоритми сортування на масивах різного розміру

## Висновок
У ході виконання практичної роботи було створено вебзастосунок для управління студентами. Було реалізовано основні операції роботи з масивами: додавання, видалення, редагування, фільтрацію, сортування та обчислення статистики. Також було реалізовано алгоритми Bubble Sort і Quick Sort та виконано їх порівняння за часом роботи і кількістю порівнянь. У результаті було закріплено практичні навички роботи з масивами та алгоритмами обробки даних у JavaScript.
